# hanamilanov.com

Personal academic website for **Prof. Hana Milanov, Ph.D.** — Professor of Entrepreneurship, IMD Lausanne.

Built to host on GitHub Pages, then transfer the repo to Hana's GitHub account. Custom domain: **hanamilanov.com** (fallback `hana-milanov.com`).

## Current stage: 3 design options to choose from

Open `index.html` in a browser, then click through:

| File | Direction | Feel |
|------|-----------|------|
| `option-a-minimal.html`   | **Modern Minimal** | Clean sans-serif, big hero photo, gold accent. Closest to the brief. |
| `option-b-editorial.html` | **Editorial / Serif** | Typography-led, narrow column, warm paper. Classic academic. |
| `option-c-bold.html`      | **Bold / Narrative** | Big type statements, strong colour, story-first. (Vendraminelli-inspired.) |

All three are single self-contained HTML files with the same placeholder content and her real photo. Once one is chosen, it becomes the real multi-page site.
